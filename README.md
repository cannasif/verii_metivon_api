# V3RII Metivon API

Metivon ERP'nin modüler .NET API projesidir.

## Mimari sözleşme

Yeni veya değiştirilen her modül [V3RII Senior Grade Proje Kuralları](docs/V3RII_SENIOR_GRADE_PROJECT_RULES.md) belgesine uymalıdır. Belge API ve Web için tek kanonik yapıyı, modül sınırlarını, mapping/validation/localization kurallarını, testleri ve kalite kapılarını tanımlar.

## Temel komutlar

```bash
dotnet restore
dotnet build verii_metivon_api.sln
dotnet test verii_metivon_api.sln
```
